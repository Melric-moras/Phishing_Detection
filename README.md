# Phishing Detection Web App

This is a simple full-stack web application that helps users check whether a URL is safe or potentially a phishing link.

The project is built with a basic frontend and a Node.js backend. It demonstrates how a user input can be sent to a server, processed, and returned with a result.

---

## What this project does

- Takes a URL as input from the user
- Sends it to the backend
- Checks whether the URL looks safe or suspicious
- Displays the result clearly on the screen
